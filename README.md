# 家族树 (FamilyTree)

一个用于收集和展示家族信息的家谱微信小程序。

## 🌟 项目介绍

家族树是一个专为家族设计的微信小程序，帮助家族成员：
- 📋 登记个人信息，建立家族档案
- 🌳 可视化展示家谱关系（树形图/脑图）
- 🔍 便捷查找家族成员信息
- 📱 随时随地查看家族历史

## 🎯 核心功能

1. **个人信息登记** - 首次使用填写基本信息和家族关系
2. **家谱可视化** - 树形视图、脑图视图、列表视图
3. **成员管理** - 添加、编辑家族成员信息
4. **智能搜索** - 快速查找家族成员
5. **数据统计** - 家族人数、辈分分布、生日提醒

## 🛠️ 技术栈

- **前端**: 微信小程序 / Taro
- **后端**: 微信云开发 (Node.js + MongoDB)
- **可视化**: D3.js / AntV G6
